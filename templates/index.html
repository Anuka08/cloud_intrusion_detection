Cloud Simulation with SFDO-DRNN

Advanced Cloud Security Analysis using SailFish-Dolphin Optimization and Deep Recurrent Neural Networks

About This Application

This is a research implementation that combines cloud simulation (CloudSim) with advanced machine learning techniques including:

  • SFDO: SailFish-Dolphin Optimization Algorithm
  • DRNN: Deep Recurrent Neural Networks
  • FCM: Fuzzy C-Means Clustering
  • SVM: Support Vector Machines
  • ANFIS: Adaptive Neuro-Fuzzy Inference System

The system analyzes Bot-IoT network traffic data for enhanced cybersecurity in cloud environments.
